Фильтры в Jinja2

Исходная фраза с тегами:

Фраза: {{ phrase }}

Фильтр title (слова с заглавной буквы):

Фраза: {{ phrase|title }}

Фильтр striptags (удаление HTML-тегов):

Фраза: {{ phrase|striptags }}

Фильтр safe (разрешение HTML-тегов):

Фраза: {{ phrase|safe }}

Комбинация фильтров striptags + reverse:

Фраза: {{ phrase|striptags|reverse }}

Другие полезные фильтры:

Upper: {{ phrase|striptags|upper }}
Lower: {{ phrase|striptags|lower }}
Capitalize: {{ phrase|striptags|capitalize }}
Length: {{ phrase|striptags|length }} символов
Wordcount: {{ phrase|striptags|wordcount }} слов

Комбинация нескольких фильтров (пример из задания):

striptags + replace + upper:
{{ phrase|striptags|replace('сколько', 'МНОГО')|replace('нам', 'ТУТ')|upper }}
